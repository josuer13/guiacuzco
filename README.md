## Nota importante: correr solo newIndex.hmtl
newIndex.html es la página auténtica con JS, index, sitios, platos, historia y reseñas.html son versiones separadas que funcionan mal al integrar la función de cuentas, ya que si creas tu cuenta en index.html al irte a sitios.html se borra tu cuenta.
newIndex.html las reúne y unífica, usando JS para modificar el main con solo un html, la versión pasada cambiaba de página a través del sidebar.
